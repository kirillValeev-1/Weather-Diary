## Описание программы
**Weather Diary** – это простое desktop-приложение для ведения ежедневных записей о погоде. Программа позволяет добавлять, просматривать, фильтровать записи и сохранять их в JSON-файл.

## Возможности программы
- Добавление записей о погоде с указанием даты, температуры, текстового описания и признака осадков
- Просмотр всех записей в виде таблицы
- Фильтрация записей:
  - по точной дате (формат ГГГГ-ММ-ДД)
  - по температуре – показать все дни, где температура выше заданного порога
- Удаление записей из списка
- Автоматическое сохранение данных в JSON-файл (`weather.json`)
- Загрузка ранее сохранённых записей при запуске
- Валидация вводимых данных с понятными сообщениями об ошибках

## Проверка корректности ввода
- **Дата** – должна быть в формате `ГГГГ-ММ-ДД` (например, `2026-04-15`). При неверном формате или невозможной дате появится ошибка.
- **Температура** – должна быть числом (допускаются отрицательные и дробные значения, например `-5.5`).
- **Описание** – не должно быть пустым. Минимальная длина не проверяется, но поле обязательно для заполнения.
- **Осадки** – флажок (да/нет), всегда имеет значение (по умолчанию «нет»).
